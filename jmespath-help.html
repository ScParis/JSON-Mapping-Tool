Começando com JMESPath

JMESPath é uma linguagem de consulta para JSON que permite extrair e transformar elementos de documentos JSON de forma poderosa e concisa.

Exemplo Básico - Acessar Propriedades
JSON de Entrada:
{ "pessoa": { "nome": "João", "idade": 30, "cidade": "São Paulo" } }
Expressão JMESPath:
Resultado:
"João"
Exemplo Avançado - Arrays e Filtros
JSON de Entrada:
{ "produtos": [ {"nome": "Notebook", "preco": 3500, "categoria": "eletronico"}, {"nome": "Mouse", "preco": 120, "categoria": "acessorio"}, {"nome": "Teclado", "preco": 200, "categoria": "acessorio"} ] }
Expressão JMESPath:
Resultado:
{"nome": "Mouse", "preco": 120, "categoria": "acessorio"}
Exemplo Intermediário - Funções
JSON de Entrada:
{ "numeros": [1, 2, 3, 4, 5], "texto": "JSON Mapping Tool" }
Expressão JMESPath:
Resultado:
{"soma": 15, "upper": "JSON MAPPING TOOL"}
Exemplo Avançado - Projeção com Filtro
JSON de Entrada:
{ "usuarios": [ {"id": 1, "nome": "Ana", "idade": 25, "ativo": true}, {"id": 2, "nome": "Bruno", "idade": 30, "ativo": false}, {"id": 3, "nome": "Carla", "idade": 28, "ativo": true} ] }
Expressão JMESPath:
Resultado:
["Ana", "Carla"]

Conceitos Importantes

  • Acesso a propriedades: Use ponto (.) para acessar propriedades aninhadas
  • Arrays: Use colchetes [] para acessar elementos e aplicar filtros
  • Filtros: Use [?condição] para filtrar elementos de arrays
  • Projeções: Use [*] para criar arrays a partir de propriedades
  • Funções: Use funções como sum(), length(), upper_case()
  • Multi-seleção: Use {prop1: expr1, prop2: expr2} para criar objetos

Referência Rápida JMESPath

Acesso Básico
pessoa.nome
Acessa propriedades aninhadas usando notação de ponto.
Indexação de Arrays
items[0]
Acessa elementos específicos de arrays usando índices.
Projeção de Arrays
items[*].nome
Cria array com valores de propriedades de todos os elementos.
Filtros
items[?preco>100]
Filtra elementos baseados em condições.
Multi-seleção
{nome: nome, idade: idade}
Cria novo objeto com propriedades selecionadas.
Funções - Array
length(items)
Retorna o tamanho de arrays ou strings.
Funções - Matemáticas
sum(preços)
Calcula soma de valores numéricos.
Funções - String
upper_case(texto)
Converte strings para maiúsculas/minúsculas.
Pipe
items | [0]
Encadeia múltiplas operações.
Slice
items[1:3]
Extrai subconjunto de arrays (slice).
Flatten
[][]
Achata arrays aninhados.
Sort
sort_by(items, &idade)
Ordena arrays por propriedades.
